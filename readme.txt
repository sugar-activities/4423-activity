hMouse
Programa para controlar el mouse de la XO con la cabeza.

El control debe realizarse con un puntero LED de color blanco. Próximamente se agregará información sobre el prototipo que utilizamos a este fin en la Fundación Teletón y un manual de usuario para el mejor manejo del programa.
